Use this simple 'learn Japanese' plugin for your travel website or language school website.

== Description ==

Introducing the 'Learn Japanese' plugin, an essential tool for your travel or language school website. This simple yet powerful plugin will allow your website visitors to learn basic Japanese phrases and vocabulary, making their travel experience smoother and more enjoyable.

Our plugin is designed to help web developers who want to learn Japanese. It's effortless to install, and you can use it right away! 

With our plugin, you'll be able to:
* Learn Japanese quickly and easily with our quizzes!
* Gain access to a wide range of Japanese phrases that are useful for travelling or website development.
* Enjoy a user-friendly interface designed with simplicity in mind!
* And it is totally free!

Use the following shortcode to display on frontend: `[simplelearnjapanesequiz_display_form]` 
Don't miss out on this great opportunity to learn Japanese!

[Tourhand](https://tourhand.com "Tourhand Traveller Website") is a handy website for travellers traveling to Japan or Singapore. Visit us to learn more! 

== Installation == 
To install, you just need to follow these steps 
1. Upload the zip file to the `/wp-content/plugins/` directory 
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Navigate to the Settings page and configure.
